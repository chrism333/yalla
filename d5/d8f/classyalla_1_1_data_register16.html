Detailed Description

template<addr_t addr>
class yalla::DataRegister16< addr >

Simple wrapper for a IOMMPtr. Represents a 16-bit io data register.

In contrast to a Register a DataRegister16 is represented only by a 16-bit value and is not a set of individual bits.

Template Parameters
addrAddress of the register.

static INLINE uint8_t yalla::DataRegister16< addr >::read ( )
inlinestatic

Read a 16-bit integer from the register.

This method always disables interrupts and restores status of the interrupt flag after the read operation is performed. If interrupts are always disabeled when this method is called, read_unsafe() can be used for more efficient read access.

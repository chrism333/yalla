Detailed Description

template<addr_t addr>
class yalla::DataRegister< addr >

Simple wrapper for a IOMMPtr. Represents a 8-bit io data register.

In contrast to a Register a DataRegister is represented only by a 8-bit value and is not a set of individual bits.

Template Parameters
addrAddress of the register.

Detailed Description

template<typename T, addr_t addr>
class yalla::IOMMPtr< T, addr >

A static pointer that provides read and write access to the I/O Memory Space.

When reading or writing types with size greater than 1 byte interrupts have to be disabled to guarantee atomic access.

Template Parameters
Ttype that is used for read and write access
addrthe address this pointer points to (absolute memory address)
